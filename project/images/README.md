# images/ — 補圖資料夾

把題目的圖檔放在這個資料夾，放進去就會自動顯示，不需要改任何程式。

## 命名規則（檔名 = 原始題號）
題庫每題的原始題號格式為  {科目代碼}-{工作項目}-{題序}  ，例如 12500-01-3。

- 題幹圖（符號在題目敘述裡，1 張）：  {題號}.png        例：12500-01-3.png
- 選項圖（四個答案都是圖形，4 張）：  {題號}-1.png ~ {題號}-4.png   例：12500-01-6-1.png

## 建議規格
- 格式：PNG（去背或白底皆可；程式會自動鋪白底以相容深淺主題）
- 題幹圖：寬 320–640px
- 選項圖：約 400×400px（正方）
- 單檔 < 100KB

## 哪些題要補？
打開專案根目錄的「image-checklist.html」，裡面列出全部 48 題含圖題目、原始題號、題目敘述、以及該放的精確檔名，並會即時預覽是否已上傳。
